A propos

Je suis Abdoul Kader Diomandé, étudiant en droit et informatique.
Je suis un développeur web passionné par la création de solutions innovantes.
L'informatique est pour moi un domaine fascinant.
Ça l'est devenu encore après ma découverte du développement web.

c'est ainsi que commence mes premier pas vers le développement web.

C'est quoi le développement web ?

Le développement Web est le travail nécessaire au développement d'un site Web pour Internet ou un intranet.
